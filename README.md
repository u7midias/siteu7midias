# Site U7 Mídias

Site estático em HTML, CSS e JavaScript.

## Como publicar rápido

1. Entre no Netlify ou Vercel.
2. Faça upload da pasta `u7-site`.
3. Conecte o domínio `u7midias.marketing`.
4. Depois troque o e-mail para o profissional quando estiver pronto.

## Arquivos

- `index.html`: estrutura do site
- `styles.css`: visual completo
- `script.js`: animações e formulário para WhatsApp
